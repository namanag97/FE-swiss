03 — Body

Body Type

Information density is not about cramming. It's the clarity of relationships between data points that creates understanding.

Standard interface text at the default body size. Used for form content, descriptions, and general reading.

body/default — Geist 400 / 14px / lh normal / primary

Long-form prose with relaxed leading for comfortable reading. Bold emphasis lifts to weight 500 and primary color. Paragraphs, documentation, blog articles — any sustained reading uses this height.

body/prose — Geist 400 / 14px / lh relaxed / secondary. Strong: primary + 500

Emphasis rendered in italic. Same metrics as prose — used for quotations or soft emphasis within body copy.

body/emphasis — Geist 400 / 14px / italic / lh relaxed / secondary

Body text with an inline link embedded. Weight 500, underline offset 3px, decoration w300 → w900 on hover.

body/link — Geist 500 / 14px / underline / lh relaxed / primary / decoration: default → strong

Deprecated or removed content rendered with strikethrough and placeholder color.

body/strikethrough — Geist 400 / 14px / line-through / lh relaxed / placeholder

09 — Surfaces

Background Materials

surface/ground — w100 + paper texture 4%

GROUND — #f5f4f0 + NOISE 4%
Page substrate. fractalNoise: baseFreq 0.65, octaves 4, stitch. Fixed overlay, pointer-events none, z-index 10000

surface/primary — w0 + elevation/subtle

PRIMARY — #fefdfb + SHADOW
Content panels, cards. Shadow: 0 1px 2px rgba(26,25,20,0.04), 0 8px 32px rgba(26,25,20,0.06)

surface/subtle — w50

SUBTLE — #faf9f6
Card headers, input backgrounds, code blocks. No shadow, border only

surface/inverse — w900 + paper texture 3%

INVERSE — #1a1914 + NOISE 3%
Dark panels, toasts, tooltips, hero. Same texture, reduced to 3% opacity

surface/glass — rgba + backdrop blur

GLASS — RGBA(250,249,246,0.18) + BLUR 12PX
Floating headers, scroll spy. backdrop-filter: blur(12px) brightness(1.15) saturate(1.1). Border: rgba(255,253,251,0.12)
